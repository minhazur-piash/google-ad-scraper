### How to run
##### Google 
> command format:
`uv run main.py google domain flag`
`uv run main.py fb slug flag`

> example command : `uv run main.py google hoski.ca 1`
> example command : `uv run main.py fb Squarespace 1`

#### Possible flags:
```
GET_STATS = 1
GET_CREATIVES = 2
DOWNLOAD_CREATIVES = 3
```
